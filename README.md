# Ground Zero Website

This project contains the website for [GroundZero](https://x.com/groundzero_twt). It's your friendly neighborhood creative space shaping the frontier of tech, with occasional conversations and notes.

This project was also made as a submission for [Hack Club / Midnight](https://midnight.hackclub.com)

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS

## How to run?
- Clone the repo `git clone https://github.com/shashwtd/groundzero-website`
- Go to project folder `cd groundzero-website`
- Install dependencies `npm i`
- Add environment variables to .env.local file `YOUTUBE_API_KEY=YT_KEY_HERE`
- Run the app `npm run dev`
